  What is it?

  myanimelist-api-wrapper is, like the name states,
  a api wrapper, which is intended to make the task
  of querying the MAL(my anime list) api easier. Any
  endpoint that you can be queried is available 
  through the library.
	

  Why?


  •  I worked with the MAL api for my last project
AnimeLazer


  •  There weren't any recent packages for the MAL 
api



  Challenges

  The hardest challenge was figuring out how to structure a
  package. I had no prior knowledge, so I relied heavely on
  the alphavantage wrapper's project structure. It was also 
  the first time I had to deal with different module patterns.


  Main Takeaways


  • Figuring the structure of the library is one of the most 
important things.


  • Choose a module pattern beforehand.
